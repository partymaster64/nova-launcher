Everything you need

Nova Launcher packs every feature a Minecraft player needs into a sleek, native app.

🔐

Microsoft Login

OAuth2 device-code flow with credentials stored securely in your OS native keystore — never in plain text.

📦

Instance Management

Run multiple isolated Minecraft instances with their own versions, mods, saves and settings. Switch instantly.

🌍

Modrinth Integration

Browse, install and update mods, modpacks and resource packs from Modrinth without leaving the launcher.

🧑‍🎤

Skin Manager

Live 3D preview of your Minecraft skin powered by skinview3d. Upload and switch skins with one click.

🎮

Discord Rich Presence

Automatically shows what you are playing in your Discord status whenever a game session is active.

🔄

World & Save Sync

Sync saves, resource packs and screenshots across all instances via smart symlinks. One world, every instance.

☕

Java Management

Auto-detects installed Java versions and downloads the right JRE for each Minecraft version automatically.

🌐

Multilingual

Available in German, English and French. Add more languages via simple JSON locale files.

⚡

Native Performance

Built with Tauri 2 and Rust. Minimal memory footprint, instant startup — no Electron overhead.

Download

Available as a native installer for all major platforms. No Java required to run the launcher itself.
