Nova Launcher

A fast, beautiful Minecraft launcher. Manage instances, browse Modrinth, launch any version — all in one place.

↓ Download View on GitHub

Everything you need

🔒

Microsoft Login

Secure OAuth2 device-code flow. Credentials stored in your OS keystore — never in plain text.

📂

Instance Management

Multiple isolated instances with their own versions, mods and settings. Switch in one click.

🌍

Modrinth Integration

Browse and install mods, modpacks and resource packs directly from Modrinth without leaving the launcher.

👾

Skin Manager

Live 3D preview of your skin powered by skinview3d. Upload and switch skins on the fly.

🎮

Discord Rich Presence

Automatically shows what you are playing in your Discord status while a game is running.

🌐

Multilingual

Available in German, English and French. More languages can be added via simple JSON locale files.
